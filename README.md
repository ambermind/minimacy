# minimacy-distrib

Minimacy is an open-source minimalist computation system based on the principle “Less is more”.
It is designed and programmed by [Sylvain Huet](https://sylvain-huet.com/).

- Website: [https://minimacy.net/](https://minimacy.net/)
- Why: [https://minimacy.net/dok/why.php](https://minimacy.net/dok/why.php)
- Documentation and Compiling: [https://minimacy.net/dok/tutorial.php](https://minimacy.net/dok/tutorial.php)
